## 说明
* 本站的产品图使用mock.js技术来模拟数据请求效果。
* 本站的轮播图、lazy-load使用的是mint-ui插件；scroll滚动使用的是better-scroll插件。
* 数据状态的处理使用了vuex，比如支付页与我的订单页面之间的数据通信， 页面刷新会清空。
* 登录账号信息使用本地存储。
